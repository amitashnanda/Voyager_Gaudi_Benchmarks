================================================================================
METHOD NOTES
================================================================================
Sensitivity Analysis Method: PMPQ
  PMPQ (Pruning-based Mixed-Precision Quantization) computes
  layer sensitivity as the accuracy drop on BoolQ TRAIN split
  when a fixed percentage of the smallest-magnitude weights in that
  layer are set to zero. All other layers remain unchanged.
  sensitivity[i] = baseline_accuracy - pruned_accuracy.
  Layers with larger accuracy drops are more sensitive and receive
  higher bit-widths in mixed-precision quantization.
  Calibration data: Phase 1 training split of BoolQ.

Quantization: REAL (weights stored in FP16 or INT8 on GPU)
  FP16 layers: weight stored as torch.float16.
               A100 Tensor Cores run FP16 matmul natively.
  INT8 layers: weight stored as torch.int8 (1 byte per weight).
               On-the-fly dequantization: int8 * scale -> fp16.
               Then fp16 matmul on A100 Tensor Cores.
               This is weight-only quantization, same as GPTQ.

Group-Wise Quantization:
  group_size=128: every 128 consecutive weights
  share one FP16 scale factor. Same approach as GPTQ. Provides
  finer precision than a single global scale per layer.

Dataset: BoolQ Validation Set
  Binary reading comprehension benchmark (yes/no questions).
  Evaluation: log-likelihood scoring of 'yes' and 'no' tokens
  given the passage + question prompt. Model picks the token
  with the higher log-likelihood.
  Metrics: Accuracy, Precision, Recall, F1 (positive class = True).

Multi-GPU:
  4 A100 GPU(s) via nn.DataParallel.
  Quantization on CPU then model moved to GPU(s).
  DataParallel applied AFTER quantization (required order).
  Eval batch_size=64 split across all GPUs per forward pass.
  FP32 computation (no autocast) for consistency across scripts.

Clustering: percentile with 3 groups
  Cluster 0 (rank 0, bits=16 [FP16]): 7 layers, mean sensitivity=0.0098
  Cluster 1 (rank 1, bits=8 [INT8]): 7 layers, mean sensitivity=-0.0033
  Cluster 2 (rank 2, bits=8 [INT8]): 8 layers, mean sensitivity=-0.0134
